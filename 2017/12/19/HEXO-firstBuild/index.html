既然是第一次使用markdown来写东西，那就先说说搭建hexo博客的过程。

  • git环境
  • hexo环境

  • hexo 基本操作

    首先是git的环境，我们首先需要注册一个github账号，然后在本机上完成git环境的配置，下载并安装git bash
    完成以后，配置本机到github上的连接

    hexo环境，打开git bash，在命令行中使用 npm i -g hexo ,第一次安装完成后，初始化 hexo init
    之后就是在GitHub上搭建，创建。repo，名字为 yourname.github.io
    之后在gitbash中配置GitHub的帐户名，完成后修改 config.yml文件,修改如下

    deploy: type: git repo: https://github.com/YourgithubName/YourgithubName.github.io.git branch: master
    完成上述步骤后，依次通过下面三个命令
    hexo clean hexo generate hexo server

    以后每次修改博客都输入下面三个命令
    hexo clean hexo generate hexo deploy

    到这里为止，基本的搭建就完成了，非常简单，blog默认主题为landscape，修改主题的话，先得 git其他主题的git文件，
    git clone https://github.com/SuperKieran/TKL.git
    这样我们的blog就会下载主题文件，之后只需要
    修改config文件里的theme名称就可以了，主题的详细配置每个主题都不一样，需要按照它们自己的说明来。

    新建，修改博客文章的话，用markdown格式完成，随后放到posts文件夹中，更新博客就可以了
